================================================================================
                         GETTING STARTED COMMANDS
================================================================================

1. Install Dependencies:
   npm install

2. Set Up Environment:
   cp .env.example .env.local
   # Update .env.local with your credentials

3. Set Up Database:
   npm run db:push
   npm run db:seed

4. Start Development:
   npm run dev

5. Open Application:
   http://localhost:3000
